# 06. Error Codes 错误码说明

## 1. 通用响应格式

```ts
interface ApiResponse<T> {
  code: number
  message: string
  data: T
}
```

成功时：

```json
{
  "code": 0,
  "message": "success",
  "data": {}
}
```

失败时：

```json
{
  "code": 40001,
  "message": "未登录或登录已过期",
  "data": null
}
```

---

## 2. 成功码

| code | message | 说明 |
|---:|---|---|
| 0 | success | 请求成功 |

---

## 3. 鉴权错误

| code | message | 说明 |
|---:|---|---|
| 40001 | 未登录或登录已过期 | Token 缺失、错误或过期 |
| 40005 | 无权限访问该资源 | 访问不属于自己的项目或场次 |

---

## 4. 参数错误

| code | message | 说明 |
|---:|---|---|
| 40002 | 参数校验失败 | 请求字段缺失或格式错误 |
| 40003 | 邮箱已被注册 | 注册邮箱重复 |
| 40004 | 邮箱或密码错误 | 登录失败 |

---

## 5. 项目相关错误

| code | message | 说明 |
|---:|---|---|
| 40401 | 项目不存在 | 项目不存在或不属于当前用户 |
| 40402 | 场次不存在 | 场次不存在或不属于当前用户项目 |
| 41001 | 小说文本过短 | 不满足 3 章或最低字数要求 |
| 41002 | 项目缺少小说文本 | 项目 sourceText 为空 |
| 41003 | 项目状态不允许操作 | 当前状态不能执行生成或编辑 |

---

## 6. AI 与点数错误

| code | message | 说明 |
|---:|---|---|
| 50001 | AI 点数不足 | 用户点数不足以执行当前 AI 操作 |
| 51001 | AI 生成失败 | 模型调用失败 |
| 51002 | AI 返回格式解析失败 | 返回内容不是合法 JSON 或缺少 scenes |
| 51003 | AI 生成结果为空 | AI 没有返回有效场次 |

---

## 7. 系统错误

| code | message | 说明 |
|---:|---|---|
| 50000 | 服务器内部错误 | 未知异常 |
| 50002 | 数据库操作失败 | 数据库异常 |

---

## 8. 前端处理建议

| 错误类型 | 前端处理 |
|---|---|
| 40001 | 清除 token，跳转登录页 |
| 40002 | 在表单中提示用户修正 |
| 40401/40402 | 提示资源不存在，返回项目列表 |
| 50001 | 提示点数不足，阻止操作 |
| 51001/51002 | 提示生成失败，允许重试 |
| 50000 | 展示通用错误提示 |
