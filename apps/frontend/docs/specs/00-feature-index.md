# Feature Spec Index

## MVP 功能规格列表

| 文件 | 功能模块 | 优先级 | 说明 |
|---|---|---:|---|
| `01-auth.feature.spec.md` | 用户系统 | P0 | 注册、登录、获取当前用户、退出登录、路由鉴权 |
| `02-project-management.feature.spec.md` | 项目管理与 AI 生成 | P0 | 项目增删改查、小说文本保存、AI 生成剧本 |
| `03-project-editor.feature.spec.md` | 项目编辑器 | P0 | 左侧场次列表、右侧剧本编辑器、保存场次内容 |

## 今日不做内容

角色工作台、Agent 对话、AI 建议、YAML/TXT 导出、DOCX/PDF 导出、会员套餐、人物关系图、历史版本、自动保存、富文本编辑器。

```txt
注册/登录
↓
进入项目列表
↓
创建项目
↓

↓

↓
进入项目编辑页
↓
左侧选择场次
↓
右侧编辑剧本文本
↓
保存场次内容
```
